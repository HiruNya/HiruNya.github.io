A character that is used to spawn an entity onto the stage.

Fields

default: String

The default state that the entity will have upon being spawned.

state_map: HashMap<String, Arc<G2dTexture>>

A collection of all the different states and their corresponding textures.

size: [f64; 2]

The size of the images [Width, Height]

offset: Pos

The offset (in percentage) of the centre of the images. This is helpful when you want to move the entity. x = 0., y = 0. : Top-Left corner x = 0.5, y = 0.5 : Centre of the image
