Represents an action dictated by the script

Variants

Dialogue(String, String)

A dialogue that consists of the name of the speaker and the text. (Name, Text)

DialogueContinue(String)

Only changes the text in the textbox. Doesn't change the speaker.

Show(String, Option<String>)

Shows an entity on the screen and if the state is not None will change that entity's state to the state named. (Entity, Some(State))

Hide(String)

Hides an entity using its name

Spawn(String, Option<String>, Option<(f64, f64)>)
